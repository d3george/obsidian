# organize-note

将笔记整理到正确的目录并添加适当的标签。

## 使用场景
- 清理 Inbox 中的笔记
- 整理未分类的笔记
- 确保笔记符合 PARA + Zettelkasten 结构

## 分析维度
1. **内容类型**：判断是概念、摘要、代码片段还是其他类型
2. **状态**：判断笔记的完成状态
3. **来源**：识别信息来源
4. **主题**：识别主要主题和领域

## 操作建议
1. **移动位置**：
   - 概念 → `Resources/Topics/[领域]/`
   - 摘要 → `Resources/Sources/[类型]/`
   - 项目相关 → `Projects/[项目名]/`
   - 领域相关 → `Areas/[领域]/`

2. **添加标签**：
   - 根据内容类型添加 `type/` 标签
   - 根据状态添加 `status/` 标签
   - 根据来源添加 `source/` 标签

3. **创建链接**：
   - 链接到相关概念笔记
   - 添加来源链接（如果是提炼的概念）

## 输出格式
提供：
1. 建议的目标目录
2. 建议添加的标签
3. 建议创建的链接
4. 是否需要提炼概念（如果是摘要笔记）

## 注意事项
- 不要重复目录名作为标签
- 确保笔记放在最合适的位置
- 如果是摘要笔记，建议提取其中的概念
