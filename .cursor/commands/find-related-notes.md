# find-related-notes

查找与当前笔记或指定主题相关的笔记，并建议链接关系。

## 使用场景
- 想要发现知识库中与某个主题相关的所有笔记
- 想要建立笔记之间的关联
- 想要找到可能重复或冲突的概念

## 搜索策略
1. **标签匹配**：查找具有相同或相关标签的笔记
2. **关键词匹配**：在标题和内容中搜索相关关键词
3. **目录结构**：查找同一目录下的相关笔记
4. **链接分析**：查找已存在的链接关系

## 输出格式
返回：
1. 直接相关的笔记列表（高相关性）
2. 间接相关的笔记列表（中等相关性）
3. 建议添加的链接关系
4. 可能重复的概念笔记（需要合并）

## 注意事项
- 优先在 `Resources/Topics/` 中查找概念笔记
- 考虑跨目录的关联（使用标签）
- 识别可能的重复内容
