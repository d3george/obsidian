# extract-concepts-from-note

从一篇来源笔记（summary）中提取概念，并创建对应的原子化概念笔记。

## 使用场景
当你有一篇摘要笔记（如读书笔记、视频总结），需要将其中的核心概念提炼成独立的原子笔记时使用。

## 工作流程
1. 分析当前笔记，识别其中的核心概念
2. 为每个概念创建独立的笔记（如果不存在）
3. 在概念笔记中添加来源链接
4. 在原笔记中添加指向概念笔记的链接

## 输出要求
- 列出所有识别到的概念
- 为每个概念创建或更新笔记
- 确保概念笔记包含：
  - 清晰的定义（用自己的话）
  - 核心要点
  - 来源链接
  - 适当的标签

## 注意事项
- 只提取真正的概念，不要提取过于具体的内容
- 如果概念笔记已存在，更新它而不是创建重复的
- 保持概念的原子性（一个笔记一个概念）
