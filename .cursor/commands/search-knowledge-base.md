# search-knowledge-base

在知识库中搜索信息，理解上下文并回答问题。

## 使用场景
- 回答关于知识库内容的问题
- 查找特定主题的信息
- 理解概念之间的关系
- 提供学习建议

## 搜索策略
1. **语义搜索**：理解问题的意图，搜索相关内容
2. **标签过滤**：使用标签缩小搜索范围
3. **目录导航**：根据目录结构定位相关领域
4. **链接追踪**：通过双向链接发现相关内容

## 回答原则
1. **引用来源**：引用具体的笔记和位置
2. **提供上下文**：解释概念在知识库中的位置
3. **建议链接**：指出相关的笔记和概念
4. **识别空白**：如果知识库中缺少相关信息，明确指出

## 输出格式
- 直接答案（如果找到）
- 相关笔记的引用
- 建议进一步阅读的笔记
- 知识库中的空白（如果存在）

## 注意事项
- 优先在概念笔记（`Resources/Topics/`）中查找
- 考虑使用标签进行跨目录搜索
- 如果信息分散在多处，整合并指出所有相关位置
