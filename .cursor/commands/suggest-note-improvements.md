# suggest-note-improvements

分析笔记并提出改进建议，使其更符合 Zettelkasten 和 PARA 方法。

## 使用场景
- 优化现有笔记的结构
- 提高笔记的可发现性和关联性
- 确保笔记符合知识库标准

## 检查维度
1. **原子性**：笔记是否专注于单一概念
2. **链接性**：是否包含足够的双向链接
3. **标签**：标签是否准确和完整
4. **结构**：笔记结构是否清晰
5. **位置**：笔记是否在正确的目录

## 改进建议类型
1. **拆分笔记**：如果笔记包含多个概念，建议拆分
2. **添加链接**：建议添加相关概念的链接
3. **完善标签**：建议添加缺失的标签
4. **改进结构**：建议优化笔记的组织结构
5. **移动位置**：如果位置不合适，建议移动

## 输出格式
- 当前笔记的评估
- 具体的改进建议
- 优先级（高/中/低）
- 实施步骤

## 注意事项
- 保持建议的实用性和可操作性
- 尊重笔记的现有风格
- 优先建议高价值的改进
