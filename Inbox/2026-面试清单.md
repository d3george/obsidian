## 一、JavaScript 底层

**参考链接 / 参考方向**

- [MDN - JavaScript 指南](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Guide)
- [JavaScript.info](https://zh.javascript.info/)（现代 JS 教程，含事件循环、原型等）
- Node 官方文档 [The Node.js Event Loop](https://nodejs.org/en/docs/guides/event-loop-timers-and-nexttick/)（六阶段与 nextTick）
- 搜索关键词：`Event Loop 宏任务 微任务`、`JavaScript 闭包 this 原型链`

### 基础

- Event Loop 的六个阶段名称及执行顺序；微任务与宏任务的常见来源及同一轮中的执行顺序。
- 闭包的定义与典型应用（防抖、节流、私有变量、模块化）。
- `this` 的绑定规则：默认、隐式、显式（call/apply/bind）、箭头函数、new。
- 原型链：`__proto__` 与 `prototype` 的区别；属性查找沿链向上的过程；`instanceof` 与 `Object.create` 的关联。

### 进阶

- 给定含 `setTimeout`、`Promise.then`、`queueMicrotask` 的代码，能正确推导执行顺序。
- Symbol 的用途（唯一键、内置 Symbol）；迭代器协议与 `for...of`；生成器与惰性求值。
- 任务队列与作业队列在规范中的关系；`requestAnimationFrame` 与渲染时机。

## 二、HTML + CSS

**参考链接 / 参考方向**

- [MDN - HTML 入门](https://developer.mozilla.org/zh-CN/docs/Learn/HTML/Introduction_to_HTML)
- [MDN - CSS 入门](https://developer.mozilla.org/zh-CN/docs/Learn/CSS/First_steps)
- [CSS 规范（W3C / WHATWG）](https://www.w3.org/TR/CSS/) 与 [Can I use](https://caniuse.com/)
- 搜索关键词：`盒模型 BFC`、`Flex Grid 布局`、`CSS 选择器优先级`、`重排重绘`

### 基础

- HTML 语义化：常用语义标签（header、nav、main、article、section、aside、footer）及选用场景；无障碍（a11y）与可访问性基础（ARIA、键盘导航、对比度）。
- 盒模型：content-box 与 border-box；块级与内联、inline-block 的区别；外边距折叠（margin collapse）。
- 选择器优先级：特异性（specificity）计算；!important 与使用建议；层叠与继承。
- 常用布局：Flex（主轴/交叉轴、flex-grow/shrink/basis、对齐）；Grid（grid-template、gap、fr 单位）及与 Flex 的选用。

### 进阶

- BFC（块级格式化上下文）的触发条件与作用（清除浮动、避免 margin 折叠、隔离布局）。
- 重排（reflow）与重绘（repaint）的触发场景；如何通过减少重排、合成层（transform、opacity）优化渲染性能。
- 响应式与适配：媒体查询、视口单位（vw/vh）、rem/em 与根字体；移动端 1px、安全区域。
- CSS 变量（自定义属性）与主题切换；现代特性：容器查询、:has()、子网格等（按面试要求掌握程度）。

## 三、TypeScript

**参考链接 / 参考方向**

- [TypeScript 官方 Handbook](https://www.typescriptlang.org/docs/handbook/)
- [Type Challenges](https://github.com/type-challenges/type-challenges)（从 Easy 到 Medium 按需刷）
- 搜索关键词：`TypeScript 条件类型 infer`、`工具类型 Partial Pick Omit`、`模板字面量类型`

### 基础

- interface 与 type 的区别及选用场景；类型合并与声明合并。
- 泛型：泛型函数、泛型接口、泛型约束；在组件与 API 类型中的常见用法。
- `keyof`、`typeof` 在类型层面的作用及与值层面的区分。

### 进阶

- 条件类型语法；能写出「根据是否为 Promise 解开类型」等实用条件类型。
- `infer` 的用法：从函数类型推断参数/返回值类型；从数组/Promise 等解出元素类型。
- 内置工具类型的实现思路：Partial、Required、Pick、Omit、Record；能手写或等价实现。
- 模板字面量类型：拼接、提取；在路由、事件名、CSS 类名等场景的推导。
- Type Challenges Medium：至少完成半数，覆盖 Readonly、DeepReadonly、函数与对象工具类型。

## 四、React

**参考链接 / 参考方向**

- [React 官方文档（react.dev）](https://react.dev/)
- [React Server Components 介绍](https://react.dev/blog/2023/03/22/react-labs-what-we-have-been-working-on-march-2023#react-server-components)
- 搜索关键词：`React Hooks 原理`、`useEffect 依赖`、`RSC 与客户端组件边界`

### 基础

- 受控组件与非受控组件的区别及适用场景；何时用 ref 暴露 DOM/实例。
- `useState` 的更新时机（批处理）；在事件处理与异步回调中的行为差异。

### 进阶

- RSC 与客户端组件的边界；哪些能力仅能在客户端使用（浏览器 API、事件、状态等）。

- `useMemo` 与 `useCallback` 的适用场景、与重渲染的关系；过度使用的性能与可读性代价。
- 能手写带缓存或防抖的自定义 Hook（如 useMemoizedFn、useDebounce、usePrevious）。
- React 与 Vue 在组件模型、状态、生命周期上的对比（便于主职 Vue 时客观表述选型）。

## 六、PostgreSQL

**参考链接 / 参考方向**

- [PostgreSQL 官方文档](https://www.postgresql.org/docs/)（索引、MVCC、事务隔离、EXPLAIN）
- [Use The Index, Luke](https://use-the-index-luke.com/)（SQL 与索引）
- 搜索关键词：`PostgreSQL MVCC`、`EXPLAIN ANALYZE 执行计划`、`B-tree GIN 索引`

### 基础

- 四种事务隔离级别及各自能避免的脏读、不可重复读、幻读。
- B-tree 索引适用的查询类型；何时会退化为 Seq Scan（无索引、函数包裹、类型不匹配等）。
- `EXPLAIN` 与 `EXPLAIN ANALYZE` 的区别；能从执行计划中判断是否走索引、代价估算是否合理。

### 进阶

- MVCC：多版本、事务 ID、可见性判断、行版本与 VACUUM 的关系。
- 联合索引的最左前缀原则；能写出会用到与不会用到该联合索引的查询例子。
- B-tree 与 GIN 的适用场景：等值/范围 vs JSONB、全文检索、数组包含等。
- VACUUM 的作用（回收死元组、更新统计信息）及其与 MVCC 的关系。
- 高并发写入下的瓶颈；分表、分区、读写分离、消息队列的选型思路。
