# Zed Vim 高频快捷键指南

> 相关概念：[[tmux-cheatsheet]]（终端复用快捷键）、[[superpowers]]（开发工作流框架，Zed 为常用编辑器）

基于 `zed-vim-keymap.json` 分类整理的高频快捷键。

## 一、基础移动（Normal 模式，日均 1000+ 次）

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `h` `j` `k` `l` | 左下上右 | 替代方向键，手不离主键区 |
| `w` / `b` | 跳到下一个/上一个单词首 | 在行内快速跳转定位 |
| `e` / `g e` | 跳到当前/上一个单词尾 | 精准停在单词末尾做操作 |
| `0` / `$` | 行首 / 行尾 | 快速移动行边界 |
| `^` | 第一个非空白字符 | 编辑代码时最常用的行首定位 |
| `g g` / `G` | 文件首 / 文件尾 | 跳到文件头尾 |
| `}` / `{` | 下一段 / 上一段 | 跨段跳跃 |
| `Ctrl-d` / `Ctrl-u` | 半页下 / 半页上 | 大文件快速浏览 |
| `Ctrl-f` / `Ctrl-b` | 整页下 / 整页上 | 整页翻页 |
| `H` / `M` / `L` | 屏幕顶 / 中 / 底 | 可视范围内精准定位 |

## 二、搜索与查找（日均 100+ 次）

| 快捷键                   | 描述           | 使用场景                      |
| --------------------- | ------------ | ------------------------- |
| `/`                   | 向下搜索         | 查找关键词                     |
| `?`                   | 向上搜索         | 反向查找                      |
| `n` / `N`             | 下一个/上一个匹配    | 循环遍历搜索结果                  |
| `*` / `#`             | 光标处词向下/上搜索   | 快速选中变量名查找所有引用             |
| `g /`                 | 全局搜索（跨文件）    | 项目中搜索关键词                  |
| `f{char}` / `F{char}` | 向前/后跳到指定字符   | 行内精准定位（如 `f)` 跳到右括号）      |
| `t{char}` / `T{char}` | 向前/后跳到指定字符前  | 行内定位但不包括该字符（如 `t)` 在右括号前） |
| `;` / `,`             | 重复/反向 f/t 搜索 | 连续跳转不用重复输入                |
| `%`                   | 匹配括号/引号      | 快速跳到配对的括号                 |

## 三、编辑操作（Normal 模式，日均 200+ 次）

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `i` / `a` | 光标前/后插入 | 最基础的进入插入模式 |
| `I` / `A` | 行首非空白处/行尾插入 | 行头添加 import 或行尾加分号 |
| `o` / `O` | 下方/上方新开一行并插入 | 在函数前后加逻辑，不破坏当前行 |
| `dd` | 删除当前行 | 删除无用行 |
| `cc` | 删除当前行并插入 | 重写一行代码 |
| `C` | 从光标处删除到行尾并插入 | 修改行后半部分 |
| `yy` | 复制当前行 | 复制一行做模板 |
| `p` / `P` | 粘贴到光标后/前 | 粘贴刚才复制的内容 |
| `u` / `Ctrl-r` | 撤销 / 重做 | 基本编辑回退 |
| `.` | 重复上次操作 | 连续做相同修改（最高效快捷键之一） |
| `r{char}` | 替换一个字符 | 修正拼写错误，如 `rt` → `ry` |
| `~` | 切换大小写 | 变量命名修改 |
| `D` | 删除到行尾 | 截断行尾多余代码 |
| `Y` | 复制整行 | 同 `yy` |

## 四、文本对象（operator + 对象，日均 100+ 次）

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `ciw` | 改当前单词 | 修改变量名 |
| `ci"` / `ci'` / `ci(` / `ci{` | 改引号/括号/大括号内内容 | 修改字符串/函数参数/代码块内容 |
| `diw` | 删除当前单词 | 删除无用变量 |
| `yiw` | 复制当前单词 | 快速复制变量名 |
| `cit` | 改 HTML 标签内容 | 修改 JSX 元素内容 |
| `dat` | 删除整个 HTML 标签 | 移除元素 |
| `viw` | 选中当前单词 | 可视化选词 |
| `vib` / `vi)` / `vi{` | 选中括号内内容 | 选中函数体/代码块 |
| `a` 前缀（如 `dab`） | around - 包含括号本身 | 删除整个括号表达式 |

## 五、Visual 模式（可视化选择）

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `v` | 字符可视化 | 精准选区 |
| `V` (Shift-v) | 行可视化 | 选中整行或多行 |
| `Ctrl-v` | 块可视化 | 列编辑（多行同时修改） |
| `>` / `<` | 缩进 / 反缩进 | 调整代码块缩进 |
| `=` | 自动格式化缩进 | 修复代码缩进 |
| `gc` | 注释 / 取消注释 | 批量注释代码块 |
| `J` | 合并行 | 合并多行为一行 |
| `U` / `u` | 大写 / 小写 | 批量转换大小写 |

## 六、Zed 专属增强

### 代码导航（g 前缀）

| 快捷键               | 描述           | 使用场景            |
| ----------------- | ------------ | --------------- |
| `gd`              | 跳转到定义        | 查看函数/变量定义       |
| `gD`              | 跳转到声明        | 查看类型声明          |
| `gy`              | 跳转到类型定义      | 查看类型定义          |
| `gi`              | 跳转到实现        | 查看接口实现          |
| `grr`             | 查找所有引用       | 了解变量/函数在哪里被使用   |
| `grn` / `space r` | 重命名          | 安全重命名变量/函数      |
| `g.` / `gra`      | 代码操作         | 快速修复/导入         |
| `g shift+a`       | 查找所有引用       | 同 `grr`         |
| `gs`              | 大纲视图         | 查看文件结构          |
| `g shift+s`       | 项目符号搜索       | 跨文件搜索符号         |
| `gx`              | 打开 URL       | 点击代码中的链接        |
| `gf`              | 打开选中文件名      | 快速打开 import 的文件 |
| `g shift+i`       | 跳转到实现        | 同 `gi`          |
| `gB`              | 查看 git blame | 查看某行修改历史        |
| `g]` / `g[`       | 上/下一个诊断      | 跳转到错误/警告        |

### Space 前缀（快捷面板）

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `space f` | 文件查找 | 快速打开项目文件 |
| `space k` | 悬停提示 | 查看类型/文档 |
| `space d` | 跳转到诊断 | 跳到下一个错误 |
| `space a` | 代码操作 | 快速修复 |
| `space c` | 切换注释 | 注释当前行 |
| `space p` | 粘贴 | |
| `space y` | 复制 | |
| `space /` | 全局搜索 | 项目搜索 |

### Git 操作

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `du` | stage 并跳转下一个 | 交互式暂存改动 |
| `dU` | unstage 并跳转下一个 | 取消暂存 |
| `dp` | 还原改动 | 丢弃未暂存的修改 |
| `do` | 切换 diff hunks | 查看代码变更 |
| `[c` / `]c` | 上/下一个 hunk | 在 git 变更间跳转 |
| `[r` / `]r` | 上/下一个引用 | 跳转引用 |

### Helix 模式增强

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `[x` / `]x` | 扩大/缩小语法选择 | 逐步选中更大的 AST 节点 |
| `space w v` / `space w s` | 垂直/水平分割窗口 | 多文件并排编辑 |
| `Ctrl-^` | 切换上一个文件 | 在两个文件间快速切换 |

## 七、窗口与标签页

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `Ctrl-w v` | 垂直分割 | 并排看两个文件 |
| `Ctrl-w s` | 水平分割 | 上下分割 |
| `Ctrl-w h/j/k/l` | 切换到相邻窗口 | 在分割窗口间移动焦点 |
| `Ctrl-w q` | 关闭当前窗口 | 关闭分割 |
| `Ctrl-w =` | 等分窗口 | 重置窗口大小 |
| `gt` / `gT` | 下一个/上一个标签页 | 标签页切换 |
| `ZZ` | 保存并关闭 | 完成编辑后关闭文件 |

## 八、折叠（代码折叠）

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `za` | 切换折叠 | 折叠/展开当前块 |
| `zc` / `zo` | 折叠 / 展开 | 手动折叠代码 |
| `zM` / `zR` | 全部折叠 / 全部展开 | 快速概览文件结构 |

## 九、其他实用快捷键

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `Ctrl-o` / `Ctrl-i` | 后退 / 前进（跳转历史） | 跳回之前的位置 |
| `Ctrl-]` | 跳转到定义 | 同 `gd`，经典 vim 方式 |
| `m{a-z}` | 设置标记 | 标记重要位置 |
| `'{a-z}` | 跳到标记 | 快速回到标记处 |
| `:` | 命令面板 | 执行任意命令 |
| `q` / `@` | 录制/回放宏 | 重复性批量编辑 |
| `Ctrl-a` / `Ctrl-x` | 递增/递减数字 | 修改版本号、序号等 |

## 十、Insert 模式专属

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `Ctrl-c` / `Esc` | 退出到 Normal | 完成编辑 |
| `Ctrl-w` | 删除前一个单词 | 快速删词 |
| `Ctrl-u` | 删除到行首 | 清空当前行输入 |
| `Ctrl-o` | 临时 Normal 模式（一次命令） | 短暂移动光标后自动回 Insert |
| `Ctrl-r` | 插入寄存器内容 | 粘贴已复制的内容 |
| `Ctrl-x Ctrl-o` | 自动补全 | 触发代码补全 |
| `Ctrl-x Ctrl-l` | 代码操作 | 快速修复 |
