# Tmux 快捷键速查表

基于 tmux key bindings 整理的高频快捷键。前缀键为 `Space`（默认 `Ctrl-b`）。

## 一、窗口管理（Space 前缀，日均 50+ 次）

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `c` | 创建新窗口 | 开一个新终端做其他任务 |
| `n` | 选择下一个窗口 | 切到下一个工作区 |
| `l` | 选择上一个当前窗口 | 在最近两个窗口间快速切换（最常用） |
| `w` | 从列表选择窗口 | 浏览所有窗口并选择 |
| `0-9` | 选择编号窗口 | 直接跳到指定编号窗口 |
| `,` | 重命名当前窗口 | 给窗口起有意义的名字（如 "dev", "server"） |
| `.` | 移动当前窗口 | 调整窗口顺序 |
| `&` | 关闭当前窗口 | 关闭无用窗口 |
| `i` | 显示窗口信息 | 查看当前窗口的进程信息 |
| `(` / `)` | 切换到前/下一个客户端 | 多终端窗口间切换 |
| `L` | 切换到最后一个客户端 | 回到上一个终端窗口 |
| `D` | 从列表选择并分离客户端 | 管理多个连接 |

## 二、面板分割与管理（Space 前缀，日均 30+ 次）

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `%` | 垂直分割窗口 | 左右并排查看两个进程 |
| `"` | 水平分割窗口 | 上面看日志，下面输入命令 |
| `o` | 选择下一个面板 | 在分割面板间移动焦点 |
| `;` | 选择上一个活动面板 | 在两个面板间来回切换 |
| `f` | 搜索面板 | 按内容查找面板 |
| `q` | 显示面板编号 | 按数字快速跳面板 |
| `x` | 关闭当前面板 | 关闭无用面板 |
| `z` | 缩放当前面板 | 临时放大面板看详情，再按一次恢复 |
| `{` | 与上方面板交换 | 调整面板布局顺序 |
| `}` | 与下方面板交换 | 调整面板布局顺序 |
| `m` | 切换标记面板 | 标记面板用于后续操作 |
| `M` | 清除标记面板 | 取消面板标记 |
| `!` | 面板拆分为新窗口 | 把面板独立成新窗口 |

## 三、布局切换

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `Space` | 选择下一个布局 | 循环切换布局 |
| M-1 | 设置水平均分布局 | 多个面板水平排列 |
| M-2 | 设置垂直均分布局 | 多个面板垂直排列 |
| M-3 | 设置主水平布局 | 一个主面板+右侧小面板 |
| M-4 | 设置主垂直布局 | 一个主面板+下方小面板 |
| M-5 | 选择平铺布局 | 所有面板网格排列 |
| M-6 | 设置主水平镜像布局 | 主面板在右 |
| M-7 | 设置主垂直镜像布局 | 主面板在下 |
| E | 均匀分布面板 | 调整面板大小使其均匀 |

## 四、面板导航（无前缀，方向键）

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| ↑ | 选择上方面板 | 切到上方面板 |
| ↓ | 选择下方面板 | 切到下方面板 |
| ← | 选择左方面板 | 切到左方面板 |
| → | 选择右方面板 | 切到右方面板 |

## 五、面板大小调整

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| M-↑ | 面板向上调整 5 格 | 快速调整面板尺寸 |
| M-↓ | 面板向下调整 5 格 | 快速调整面板尺寸 |
| M-← | 面板向左调整 5 格 | 快速调整面板尺寸 |
| M-→ | 面板向右调整 5 格 | 快速调整面板尺寸 |
| ↑ | 面板向上调整 | 微调面板尺寸 |
| ↓ | 面板向下调整 | 微调面板尺寸 |
| ← | 面板向左调整 | 微调面板尺寸 |
| → | 面板向右调整 | 微调面板尺寸 |

## 六、复制模式（Space 前缀，日均 20+ 次）

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `[` | 进入复制模式 | 查看历史输出、选择文本 |
| `]` | 粘贴最近缓冲区 | 粘贴刚才复制的内容 |
| `=` | 从缓冲区列表选择 | 选择历史剪贴板内容 |
| `#` | 列出所有缓冲区 | 查看历史复制内容 |
| `-` | 删除最近缓冲区 | 清理不需要的复制内容 |
| PPage | 进入复制模式并向上滚动 | 快速查看之前输出 |
| DC (Del) | 重置滚动，跟随光标 | 回到实时输出 |

### 复制模式内导航

进入复制模式后（`[`），支持 vim 风格导航：

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `hjkl` / 方向键 | 移动光标 | 定位要复制的文本 |
| `Space` | 开始选择 | 标记选择起点 |
| `Enter` | 结束选择并复制 | 完成复制 |
| `/` | 向下搜索 | 在输出中查找关键词 |
| `?` | 向上搜索 | 反向查找 |
| `n` / `N` | 下一个/上一个匹配 | 遍历搜索结果 |
| `g` / `G` | 跳转到开头/结尾 | 快速定位输出边界 |

## 七、会话管理（Space 前缀）

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `d` | 分离当前客户端 | 保持后台运行，断开连接 |
| `s` | 从列表选择会话 | 切换到其他 tmux 会话 |
| `$` | 重命名当前会话 | 给会话起有意义的名字 |

### 常用命令行命令

| 命令 | 描述 | 使用场景 |
|---|---|---|
| `tmux` | 创建新会话 | 启动 tmux |
| `tmux new -s name` | 创建命名会话 | 启动带名称的会话 |
| `tmux attach` | 连接最近会话 | 恢复断开的会话 |
| `tmux attach -t name` | 连接指定会话 | 恢复特定命名的会话 |
| `tmux ls` | 列出所有会话 | 查看运行中的会话 |
| `tmux kill-session -t name` | 关闭指定会话 | 结束不需要的会话 |

## 八、实用工具

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| `:` | 命令提示符 | 执行 tmux 命令（如 `:set -g status off`） |
| `/` | 描述按键绑定 | 查询某个快捷键的作用 |
| `?` | 列出所有键绑定 | 查看完整快捷键列表 |
| `~` | 显示消息 | 查看 tmux 通知/错误 |
| `r` | 重新绘制客户端 | 刷新显示（终端花屏时） |
| `C` | 自定义选项 | 修改 tmux 配置 |
| `t` | 显示时钟 | 暂时离开时显示时间 |
| `<` | 显示窗口菜单 | 可视化窗口操作 |
| `>` | 显示面板菜单 | 可视化面板操作 |

## 九、移动可见区域

| 快捷键 | 描述 | 使用场景 |
|---|---|---|
| ↑ | 向上移动窗口可见部分 | 滚动查看窗口上方内容 |
| ↓ | 向下移动窗口可见部分 | 滚动查看窗口下方内容 |
| ← | 向左移动窗口可见部分 | 查看窗口左侧内容 |
| → | 向右移动窗口可见部分 | 查看窗口右侧内容 |
