# Parallels 端口转发

## 定义

Parallels Desktop 虚拟机端口转发功能，将宿主机的端口映射到虚拟机的指定端口。在共享网络模式下使用。

## 核心要点

### 配置步骤

1. 右键虚拟机 → **配置** → **硬件** → **网络**
2. 点击 **端口转发** → 添加规则：
   - **名称**：自定义
   - **协议**：TCP 或 UDP
   - **主机端口**：宿主机端口
   - **虚拟机端口**：虚拟机内端口
   - **虚拟机 IP**：虚拟机内网 IP

### 常见用途

- SSH 访问：宿主机 2222 → VM 22
- 开发服务器：宿主机 8080 → VM 80
- 数据库连接：宿主机 5432 → VM 5432

### 网络模式

端口转发仅在**共享网络**模式下可用。桥接模式下虚拟机直接获取局域网 IP，无需端口转发。
